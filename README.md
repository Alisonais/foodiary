## ⚙️ Setup

1. **Clone o repositório**
  ```bash
  git clone https://github.com/seu-usuario/foodiary-api.git
  cd foodiary-api
  ```

2. **Instale as dependências**
  ```bash
  npm install
  ```

3. **Configure as variáveis de ambiente**
  - Crie um arquivo `.env` baseado no `.env.example`.

4. **Inicie o servidor**
  ```bash
  npm start
  ```
